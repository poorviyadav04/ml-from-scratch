# Simple Linear Regression

This project demonstrates **Simple Linear Regression** to model the relationship between a student's **CGPA and placement package**.

## Dataset

Student placement dataset containing:
- CGPA
- Placement package

## Workflow

1. Load dataset
2. Visualize relationship between CGPA and package
3. Train linear regression model
4. Predict package values

## Technologies

Python • Pandas • NumPy • Matplotlib • Scikit-learn
