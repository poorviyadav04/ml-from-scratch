# Retail Sales Prediction using Multiple Linear Regression

This project builds a **Multiple Linear Regression model** to predict retail store sales.

## Key Steps

- Data preprocessing
- Feature engineering
- Ordinary Least Squares (OLS) regression
- Feature significance analysis
- Multicollinearity detection using VIF
- Residual analysis

## Evaluation

Model performance is evaluated using:

- R² Score
- Mean Absolute Error

## Technologies

Python • Pandas • Statsmodels • Seaborn • Scikit-learn
