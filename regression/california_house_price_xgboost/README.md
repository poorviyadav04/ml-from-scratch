# California Housing Price Prediction using XGBoost

This project predicts housing prices using the **California Housing dataset**.

## Workflow

1. Load and explore dataset
2. Perform correlation analysis
3. Train XGBoost regression model
4. Evaluate model performance

## Evaluation Metrics

- R² Score
- Mean Absolute Error

## Technologies

Python • Pandas • NumPy • XGBoost • Seaborn • Scikit-learn
