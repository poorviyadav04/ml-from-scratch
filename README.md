# Machine Learning From Scratch

This repository contains implementations of core machine learning algorithms built from scratch using Python and NumPy.  
The goal of this project is to understand the mathematical foundations of machine learning models by implementing them manually rather than relying entirely on high-level libraries.

## Purpose of the Project

Many machine learning workflows rely heavily on high-level libraries such as Scikit-learn or TensorFlow.  
This repository focuses on implementing algorithms from first principles to better understand:

- optimization
- gradient descent
- neural network training
- evaluation metrics
- model behaviour
