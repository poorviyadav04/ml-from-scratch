# Planar Data Classification with a Neural Network

This project implements a **two-layer neural network from scratch using NumPy** to classify non-linearly separable data.

The model is compared against **Logistic Regression**, which fails to capture the complex decision boundary.

## Dataset

Synthetic planar dataset generated using NumPy.

## Implemented Components

- Logistic Regression baseline
- Two-layer neural network
- Forward propagation
- Backpropagation
- Cross-entropy loss
- Gradient descent optimization

## Experiments

The model evaluates different hidden layer sizes to observe how model capacity affects the decision boundary and classification accuracy.

## Technologies

Python • NumPy • Matplotlib • Scikit-learn • Jupyter Notebook
