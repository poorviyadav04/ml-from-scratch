# Logistic Regression Image Classifier

This project implements **Logistic Regression from scratch using NumPy** to classify images as **cat or non-cat**.

## Dataset

- Cat vs Non-Cat image dataset (.h5 format)
- Images are flattened and normalized before training

## Implemented Components

- Sigmoid activation
- Parameter initialization
- Forward propagation
- Backpropagation
- Gradient descent optimization
- Binary prediction

## Workflow

1. Load image dataset
2. Flatten and normalize images
3. Train logistic regression model
4. Evaluate accuracy on test data
5. Analyze learning curve for different learning rates

## Technologies

Python • NumPy • Matplotlib • h5py • SciPy • Jupyter Notebook
